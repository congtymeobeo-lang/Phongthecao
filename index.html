Giới thiệu

PhongCard bán giftcode/khuyến mãi. Giao diện hiện đại, vận hành mượt. Mỗi giftcode có thể là: PHONGCARD (lượt quay), PHONGVIP (bonus), PHONGGIFT (tiền trực tiếp).

Tỷ lệ vòng quay

Kết quả quay theo tỷ lệ bạn đặt:

  • 10% → 7,000 VND
  • 20% → 5,000 VND
  • 40% → Các mệnh giá khác: 2,000 / 3,000 / 4,000
  • 30% → Câu chuyện (random)

Ghi chú

Dashboard không hiển thị mã. Bạn có thể phát code riêng, gửi cho khách. Mỗi code chỉ dùng 1 lần.
